# SISYPHUS-LAB

This repository was cloned and tested from SISYPHUS-LAB.

## Machine

- Lenovo ThinkCentre M75q Gen 2 Tiny
- AMD Ryzen 7 PRO 5750GE
- 16 GB RAM
- 512 GB NVMe SSD
- Windows 11 Pro

## Purpose

SISYPHUS-LAB is the initial home lab server used for:

- GitHub workflow testing
- VS Code development
- Remote Desktop access from Mac
- Future Docker experimentation
- Future virtual machine testing
- Future AI and automation experiments

## Setup Notes

- Host machine is accessed remotely from a Mac using Windows App / Remote Desktop.
- GitHub repositories are stored under `C:\GitHub`.
- Git is configured with the `rational-absurdist` GitHub identity.
