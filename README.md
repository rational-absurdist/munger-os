# Munger OS

A Streamlit app for practical wisdom, decision journaling, mental models, bias detection, and anti-stupidity checklists.

## Philosophy

This is not a quote app. It is a judgment-training system.

Core principles:

- Avoid stupidity before chasing brilliance.
- Invert problems.
- Respect incentives.
- Stay inside your circle of competence.
- Use opportunity cost.
- Leave a margin of safety.
- Think in probabilities.
- Define disconfirming evidence.
- Let wisdom compound.

## How to Run

From the project folder:

```bash
pip install -r requirements.txt
streamlit run app.py
```
